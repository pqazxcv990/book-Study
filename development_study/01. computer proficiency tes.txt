alt +enter =셀 안에서 주을 바꿔 계속 입력하기
cntl+enter 여러셀에 동일한 내용 입력하기
범위를 지정하고 enter 누르면 지정한 범위 안에서만 셀 포인터가 이동
셀을 선택하고 ALT+↓를 누르면 같은 열에 입력된 문자열 목록이 표시된다.

자동완성 데이터 입력 중 처음 몇자가 같은열에 이미 입려고딘 내용과 동일하면 자동으로 나머지 내용이 채워진다.
(문자 데이터에만 적용되고, 숫자, 날짜, 시간 형식의 데이터에는 적용되지 않는다.)
데이터 입력 도중 입력을 취소하려면 ESC나 CTRL+Z를 누른다.

문자 데이터
문자 데이터는 한글, 영문 특수문자, 문자와 숫자가 혼합된 데이터다.
기본적으로 셀의 왼쪽으로 정렬된다.

주기억장치
ROM : 비휘발성 자가진단 프로그램이 저장됨. 
RAM : 휘발성 메모리

플래시 메모리: MP3 플레이어, 디지털 카메라,  SD카드에 이용됨
전기소모 적음 속도 빠르다

MASK ROM:   제조사에서 1회기록후 수정못함 

가상메모리: 가짜 보조기억 장치(하드디스크) 주기억 장치 처럼 사용하는 기법 

버퍼 메모리: 데이터를 주고 받을 때 속도 차이를 해결하는 임시공간 ( 복사하기 잘라내기, 붙이기 할때 사용)
연관 메모리 : 주소가 아닌 내요으이 일부를 이용해 데이터를 가져오는 방식 EX) 검색어 , 해시태그
            장점: 정보검색에 신속 단점 : 하드웨어 비용 증가
펌웨어 : 주로 ROM에 저장, 하드웨어 제어(통제)

한글 windows 10 특징

그래픽 사용자 인터페이스(gui 사용: 키보드로 명령어를 직접 입력하지 않고, 아이콘이나 메뉴를 마우스로 선택하여 모든 작업을 수행하는 사용자 작업 환경
선점형 멀티태스킹(preemptive multi-tasking :운영체제가 각 작업의 cpu 이용 시간을 제어하여 앱 실행중 문제가 발생하면 해당 앱을 강제 종료시키고, 모든 시스템 자원을 반환하는 멀티태스킹 운영방식
플러그 앤 플레이(pnp; plug&play) :컴퓨터 시스템에 하드웨어를 설치했을 때, 해당 하드웨어를 사용하는 데 필요한 시스템 환경을 운영체제가 자동으로 구성해주는 것
하드웨어 : 소프트웨어와 함께 컴퓨터를 구성하는 필수 요소중 하나. 비유하자면 인간의 몸에 해당하는 부분이기 대문에 없어서는 안되는 것이다.
기본적으로 입력장치, 출력장치, 연산장치, 기억장치, 제어장치로 나뉨.
OLE(object linking and embedding): 다른 앱에서 작성된 문자나 그림 등의 개체(object)를 현재 작성중인 문서에 장롭게 연결 (linking)하거나 삽입(embedding)하여 편집할 수 있게 하는 기능
63비트 데이터 처리 : 완전한 64비트로 데이터를 처리하므로 더 많은 양의 데이터를 빠르게 처리할 수 있으며, 사용자가 좀더 빠르고 효율 적인 시스템을 구축할 수 있게 한다.
