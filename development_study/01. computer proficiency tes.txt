스프레드시트 일반

alt +enter =셀 안에서 주을 바꿔 계속 입력하기
cntl+enter 여러셀에 동일한 내용 입력하기
범위를 지정하고 enter 누르면 지정한 범위 안에서만 셀 포인터가 이동
셀을 선택하고 ALT+↓를 누르면 같은 열에 입력된 문자열 목록이 표시된다.

자동완성 데이터 입력 중 처음 몇자가 같은열에 이미 입려고딘 내용과 동일하면 자동으로 나머지 내용이 채워진다.
(문자 데이터에만 적용되고, 숫자, 날짜, 시간 형식의 데이터에는 적용되지 않는다.)
데이터 입력 도중 입력을 취소하려면 ESC나 CTRL+Z를 누른다.

문자 데이터
문자 데이터는 한글, 영문 특수문자, 문자와 숫자가 혼합된 데이터다.
기본적으로 셀의 왼쪽으로 정렬된다.

주기억장치
ROM : 비휘발성 자가진단 프로그램이 저장됨. 
RAM : 휘발성 메모리

플래시 메모리: MP3 플레이어, 디지털 카메라,  SD카드에 이용됨
전기소모 적음 속도 빠르다

MASK ROM:   제조사에서 1회기록후 수정못함 

가상메모리: 가짜 보조기억 장치(하드디스크) 주기억 장치 처럼 사용하는 기법 

버퍼 메모리: 데이터를 주고 받을 때 속도 차이를 해결하는 임시공간 ( 복사하기 잘라내기, 붙이기 할때 사용)
연관 메모리 : 주소가 아닌 내요으이 일부를 이용해 데이터를 가져오는 방식 EX) 검색어 , 해시태그
            장점: 정보검색에 신속 단점 : 하드웨어 비용 증가
펌웨어 : 주로 ROM에 저장, 하드웨어 제어(통제)
